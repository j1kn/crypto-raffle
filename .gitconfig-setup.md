### Automatic Push (After Manual Commits)
When you run:
```bash
git commit -m "Your message"
```

It will automatically push to GitHub after the commit completes.

### Quick Push Script
When you run:
```bash
npm run push
```

It will:
1. Stage all changes (`git add -A`)
2. Commit with timestamp (`git commit`)
3. Push to GitHub (`git push origin main`)
